# Multi-Asset Crypto Trading Strategy Starter Pack

Ejecución:
    python main.py

Contenido:
- main.py        => Script principal que ejecuta las estrategias sobre varios CSVs.
- strategy.py    => Implementación de cruce de medias móviles.
- metrics.py     => Cálculo de retorno total y Sharpe ratio.
- data/          => Coloca aquí tus archivos .csv con columnas: timestamp, open, high, low, close, volume.

Requisitos:
    pip install pandas numpy matplotlib
